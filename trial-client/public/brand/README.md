# Drop your FitBridge logo here

The UI loads these files automatically (no code change needed):

- **`fitbridge-mark.png`** — the square icon (hand + dumbbell), ideally on a
  **transparent** background so it sits cleanly on the dark UI. Used in the nav,
  auth panel, onboarding, footer. Recommended: 256×256 or larger, PNG or SVG.
- `fitbridge-logo.png` *(optional)* — the full lockup (icon + "FitBridge" wordmark),
  for splash/marketing use.

Until `fitbridge-mark.png` exists, a clean SVG fallback renders in its place, so
nothing is ever broken. Add the file, commit, redeploy — done.
